# シナリオガイドライン

## 基本的な指示事項
- 全てのシナリオは日本語でMarkdown形式で記述する`.md`ファイル(以下、このmdファイルをシナリオファイルと呼びます)であること。
- 全てのシナリオファイルは各作品ディレクトリ直下のシナリオディレクトリに存在すること。

### シナリオファイルの書式
```markdown
# 第X章 第Y節 [節タイトル]

## 基本情報
- 目標文字数：[文字数]文字
- 視点： [三人称神視点 or 人物名 e.g. 田中太郎]
- 時間軸：[時間軸 e.g. プロローグから一週間後]
- 場所：[場所(移動がある場合は→で示す e.g. 大学→駐車場)]
- シーンの主目的：[シーンの主目的　e.g. 表社会への違和感と裏社会を読者に示す]

## シナリオ
[シナリオ本体]
```

### シナリオ本体の書式
シナリオを作成する際は、以下の日本の標準的な書式に従ってください。この書式は、柱、ト書、セリフという三つの主要な要素で構成されています。

#### 本文の書き方

シナリオの本文は、「柱」「ト書」「セリフ」「記号」の四つの要素で構成されますが、初心のうちは記号を意識する必要はありません。

1.  柱
    * 役割: シーンの場所と時間を指定します。
    * 書式:
        * 行の頭から詰めて書きます（字下げはしません）。
        * シーンの冒頭に丸（○）や四角（口）といった記号をつけ、続けて場所を書きます。番号は不要です。
        * 場所の指定の後にカッコ（）で時間（朝、夕、夜など）を明記します。昼の場合は特に書く必要はありません。
    * 例:
        `○片岡家の茶の間(朝)

2.  ト書
    * 役割: 登場人物の動作、出入り、情景、小道具の使い方などを記述します。小説のような心理描写や抽象的な表現は避け、あくまで具体的に書く必要があります。
    * 書式:
        * 行の頭を三文字空けて書き始めてください。
        * 複数行にわたる場合も、すべての行の頭を三文字空けて揃えます。

3.  セリフ
    * 役割: 登場人物が話す言葉です。
    * 書式:
        * まず、セリフを話す人物の名前を行の頭に書きます（字下げはしません）。
        * 男性は姓、女性は名で書くと区別がつきやすくなります。
        * 名前の次の行から、一文字下げて会話をカギカッコ「　」で囲んで書きます。
        * セリフが二行以上にわたる場合は、すべての行の頭を一文字ずつ下げて揃えます。

##### 書式の実例
```
○飛田家、茶の間
　　　畳はボロボロで、寝そべっている飛田のまわりには、破れた馬券が散らかっている。
飛田「おーい、とん子！　水、水持って来いよ！」
　　　奥から、とん子があがってくる。バケツに水が沢山入っている。
とん子「さあ、頭からおあびなさいよ」
```

## 台詞について
### I. 台詞の三大機能

全ての台詞は、以下のうち少なくとも一つ、理想的には複数の機能を同時に果たすべきです。

1.  事実を知らせる: 物語の状況、背景、キャラクター間の関係性など、観客が知る必要のある情報を伝える。
2.  心理や感情を表す: キャラクターが何を考え、何を感じているかを表現する。ただし、感情は必ずしも直接的な言葉で語られるわけではない。
3.  ストーリーを進展させる: 新たな行動を引き起こしたり、状況を変化させたりして、物語を次の段階へと進める。

### II. 優れた台詞の四つの条件

台詞を作成する際は、以下の4つの条件を常に満たすようにしてください。

1.  物語の進行方向に沿っていること: 台詞は物語の進行を助けるものでなければならず、停滞させたり、本筋から逸脱させたりしてはならない。現実の会話のように、同じことを繰り返したり、話が脱線したりするのは避けるべきです。
2.  人間の言葉であること: 不自然に美しすぎたり、詩的すぎたりする言葉ではなく、そのキャラクターが実際に使うであろう、人間味のある言葉でなければならない。
3.  魅力的であること: 観客の興味を引きつける魅力がなければならない。事務的な報告や単なる情報伝達のような無味乾燥な台詞は避ける。
4.  簡潔明瞭であること: 台詞は分かりやすく、テーマが観客に明確に伝わるものでなければならない。初心者は特に、一つの原稿用紙に一つのテーマが明確に伝わるようなセリフを選ぶべきです。

### III. 台詞の進化過程：目指すべきレベル

台詞にはレベルがあり、より高度な表現を目指すべきです。

* 避けるべき原始的な台詞:
    * 独白: キャラクターがただ状況を説明するためだけに一人で語る。
    * 「聞いたか坊主」セリフ: 観客に情報を与えるためだけに、二人のキャラクターが不自然に噂話をする。
    * 英会話セリフ: 一方が質問し、もう一方が答えるだけの、単純な問答形式。

* 洗練させるべき台詞:
    * 説明セリフ: 物語の進行に必要な説明をする台詞。これ自体は必要だが、単なる説明に終わらせず、キャラクターの感情や人間関係がにじみ出るように工夫する。

* 目指すべき高度な台詞:
    * 感情あるセリフ: 人間の感情の複雑さを表現する台詞。人間は本心と逆のことを言う場合がある（例：「あなたなんか大嫌い」と言いながら、実際は好きで仕方がない）ことを理解し、その裏腹な感情を描写する。これは、ト書きでは表現しきれない心の動きを伝える上で不可欠です。
    * 情緒あるセリフ: 最も高度な台詞。直接的な意味だけでなく、その場の雰囲気、リズム、響きの美しさで観客に訴えかける。これは「台詞のワルツ」とも言えるもので、感情のさらに奥にある「情」に訴えかける。

### IV. 台詞を使用する上での注意点

* 台詞は嘘つきである: キャラクターが口にする言葉が常に真実だとは限らない。むしろ、本心を隠すために使われることの方が多い。
* 一つのシーンにテーマは一つ: 一つのシーンの中で話があちこちに飛んではならない。そのシーンで最も伝えたいテーマに沿って台詞を構成する。
* 映像芸術の台詞であることを忘れない: 映画やテレビの台詞は、リアルなだけでなく、レンズを通して表現されることを前提としなければならない。非常に凝縮された時間の中で物語を語るため、一言一句が吟味される必要がある。

## シナリオに関する誤りのチェックリスト

以下のチェック項目を確認し、すべての項目に該当しないことを確認してください。該当する項目がある場合は、シナリオを修正する必要があります。

### セリフに関する誤り

セリフはシナリオの大部分を占めるため、その書き方には多くの注意点があります。以下の項目に該当しないか確認してください：

- [ ] 過剰セリフ: 映像を見ればわかることまで、セリフで説明しすぎてしまう誤りです。
- [ ] 説明セリフ: 物語の背景や状況を説明するためだけの不自然なセリフです。
- [ ] 英会話セリフ: 「どこへ行くのですか」「上野です」のような、不自然な問答形式のセリフを指します。
- [ ] 無性格セリフ: 登場人物の性格が全く反映されていない、誰が話しても同じようなセリフです。
- [ ] 過少セリフ: リアクションなどが不足しており、対話が不自然になる状態です。
- [ ] 抽象セリフ: 「悲しい」「美しい」といった感情や状況を、具体的な描写なしに抽象的な言葉だけで済ませてしまうセリフです。
- [ ] 文章セリフ: 話し言葉ではなく、書き言葉のような硬いセリフです。
- [ ] 告白セリフ: 人物が自らの性格や心情を直接的に語ってしまうセリフです。
- [ ] 分裂セリフ: 一つのセリフの中に二つ以上の話題が混在し、話の焦点がぼやけてしまうことです。
- [ ] 傍白・独白の乱用: 人物が心の中の思いを声に出して言う手法を使いすぎることです。
- [ ] 長セリフ: 一人の人物が延々と話し続け、聞いている側のリアクションを描写できていない状態です。

### ト書に関する誤り

ト書は、情景や人物の動作などを記述する重要な部分です。以下の項目に該当しないか確認してください：

- [ ] 過剰ト書: 物語に不必要な情報まで、細かく書きすぎてしまうことです。例えば、登場人物の服装や部屋の備品をすべて描写するなどです。
- [ ] 過少ト書: ドラマの理解に必要な人物の動作や状況描写が不足している状態です。
- [ ] 画にならないト書: 「〜と思った」のような、映像で表現できない心理描写を文章で書いてしまう誤りです。
- [ ] 描写の省略し過ぎたト書: 「何か話している」のように、具体的な内容を省略しすぎて状況が伝わらない記述です。
- [ ] 事実とフィルムとの混同: 現実で起こったことだからという理由で、ドラマ的な工夫をせずにそのまま描写してしまうことです。映像として伝わるかどうかが重要です。

---

### 時間処理についての誤り

映像ならではの時間経過の表現に関する誤りです。以下の項目に該当しないか確認してください：

- [ ] シーンのつなぎ方の誤り: 時間の経過を示すために、ただ「翌日」と書くだけで、具体的な変化を描写しないことです。また、夜のシーンから次の夜のシーンへ移るなど、時間の経過が分かりにくい構成もこれに含まれます。
- [ ] 同一シーン内の時間処理の誤り: 一つのシーンの中での時間は、基本的に現実と同じ速度で流れます。例えば、コーヒーを注文してすぐに出てくるなど、現実の時間を無視した描写は誤りです。
- [ ] 記号の乱用: F.O.（フェードアウト）やO.L.（オーバーラップ）などの撮影用語を、ドラマの区切りが曖昧なまま安易に使用してしまうことです。

---

### 人物描写に関する誤り

魅力的で一貫性のある人物を描く上での注意点です。以下の項目に該当しないか確認してください：

- [ ] 人物紹介の不足: 登場人物がどのような人物かが十分に紹介されないため、観客が感情移入できない状態です。
- [ ] 性格の不統一: 物語の前半と後半で、特に理由もなく人物の性格が大きく変わってしまうことです。
- [ ] 類型的人物: 学生は理屈っぽい、刑事は「ホシを挙げろ」と言うなど、職業や立場だけで決まりきった人物像を描いてしまうことです。
- [ ] 無性格な人物: ストーリーを進めるためだけに存在する、性格のない人形のような人物です。
- [ ] 無感情の人物: 人物が感情を失い、ストーリーの都合だけで動いてしまうことです。
- [ ] 人間関係が不明: 登場人物同士の関係性が曖昧で、観客に伝わらない状態です。
- [ ] 人物の重点のとり方の誤り: 主人公よりも脇役や端役が目立ちすぎて、物語の焦点がぼやけてしまうことです。

---

### 一般的描写の誤り

物語の構成や表現全体に関わる、より広範な誤りです。以下の項目に該当しないか確認してください：

- [ ] ひとりよがり: 作者だけがわかる専門用語や内的論理で物語を進め、観客が置いてきぼりになることです。
- [ ] 行ってこい: 本筋から外れたエピソードに長く時間を使い、再び本筋に戻ってくる構成です。物語の進行が一時停止したように感じられます。
- [ ] あと説: 事件が起きた後で、その原因を長々と説明してしまう手法です。物語の勢いを削いでしまいます。
- [ ] まぎれ: テーマや人物の性格、シーンの目的などが曖昧になり、観客が混乱することです。
- [ ] 段取り芝居: ストーリーの都合に合わせて人物を動かすため、感情や動機が不自然に見える芝居です。
- [ ] 待ってました: ある人物が、ただ他の人物の到着を待っているだけのシーンです。テンポが悪くなります。
- [ ] 団子の串ざし: 同じような盛り上がりのエピソードが連続し、クライマックスに向けた高まりが生まれない構成です。
- [ ] 聞いたか坊主: 端役の会話によって状況を説明させる安易な手法です。
- [ ] 劇薬の乱用: 回想、ナレーション、偶然の事故などを使いすぎることです。
- [ ] スタイルの不統一: シリアスなドラマの途中で突然コメディになるなど、作品全体の雰囲気が統一されていないことです。
- [ ] 葛藤の作り方の誤り: 人物個人の対立ではなく、集団の議論のようになってしまい、ドラマ性が失われることです。

## 面白さとは何か

面白いということの本質は、観客や読者に「次はどうなるのだろう？」と期待させ、物語の世界に引き込む力のことです。この期待感を創り出し、持続させることが、面白いシナリオの鍵となります。

---

## 面白くするための技術

### 省略によって面白くする4つの技術

省略は、物語のテンポを良くし、読者の想像力を刺激する重要な技術です。

1.  省略：物語の進行に不必要な部分を大胆にカットし、要点だけを見せることで、かえって物語の密度を高めます。
2.  表と裏の問題：描かれている場面（表）だけでなく、その裏で同時に進行しているであろう出来事を意識させ、どちらを描くかを選択することで、物語に深みを与えます。
3.  飛躍：時間や空間を大胆に飛び越えることで、物語に意外性とスピード感をもたらします。
4.  回想法とナレーション：過去の出来事を効果的に挿入したり、語りによって状況を説明したりすることで、物語の背景を補強し、読者の理解を助けます。

### 葛藤によって面白くする技術

葛藤（対立）は、ドラマの原動力です。

1.  人物の相克：登場人物同士の目的や価値観がぶつかり合うことで、物語に緊張感を生み出します。
2.  場面の相克：特定の状況や環境が、登場人物の目的にとって障害となるように設定します。
3.  相克の作り方：上記の人物間、あるいは人物と場面の対立を具体的にどのように作り出すかという技術です。

### 効果を高める7つの技術

物語のインパクトを強めるための具体的な手法です。

1.  効果を高めるということ：全体の見せ方や演出を工夫し、読者の感情に強く訴えかけます。
2.  構成でパンチをきかす法：物語の構成（起承転結）に工夫を凝らし、読者を飽きさせない展開を作ります。
3.  セリフを面白くする法：登場人物の性格や状況が表れる、気の利いたセリフを考えます。
4.  ト書にパンチをきかす法：登場人物の動きや情景描写を生き生きと描くことで、映像的な魅力を高めます。
5.  増幅的な表現によるパンチのつけ方：感情や状況をより強調して表現します。
6.  趣向：意表を突くアイデアや仕掛けを盛り込みます。
7.  見せ場：物語のクライマックスとなるような、視覚的に訴える華やかな場面を作ります。

### 人物の設定によるパンチのつけ方

魅力的な登場人物は、物語を面白くする上で非常に重要です。

1.  人物の魅力：読者が共感したり、興味を持ったりするような、人間味あふれる魅力的な人物を創造します。
2.  パンチのつけ方：登場人物に意外な一面を持たせたり、強い個性（キャラクター）を与えたりすることで、読者の印象に残るようにします。

### シナリオ構成の要「起承転結」

物語のテーマをドラマとして具体的に表現するための重要な枠組みである「起承転結」について解説します。これらを参考にして、脚本を作成してください。

#### 各部の機能と役割

1. 起
   * 機能: 物語の始まりであり、情勢（天）、場所（地）、人物（人）を紹介する部分です。ドラマがどのような傾向（喜劇、悲劇など）を持つのかを観客に示し、心の準備をさせる必要があります。
   * 失敗の症状: この部分が不十分だと、脚本全体が分かりにくくなります。

2. 承
   * 機能: シナリオ全体の約8〜9割を占める最も長い部分です。様々な事件や事情を積み重ねてドラマを進行させます。世阿弥の言う「破」にあたり、物語に深みを与え、詳細を描写する役割を担います。
   * 失敗の症状: この展開部が面白くないと、シナリオ全体がつまらないものになってしまいます。

3. 転
   * 機能: 物語のクライマックスにあたり、作者が最も伝えたいテーマを観客に感じさせる部分です。感動が最高潮に達する場面であり、しばしば無言の演出が効果的です。
   * 失敗の症状: この部分がうまく機能しないと、シナリオが何を言いたいのかが不明瞭になったり、理屈っぽくなり感動が薄れたりします。

4. 結
   * 機能: 物語の終わりであり、テーマを観客の心に定着させ、深い余韻を残す役割があります。ドラマ自体は「転」で完結しているため、ここは速やかに終えるべきとされています。
   * 失敗の症状: 結びがうまくいかないと、後味の悪い作品になってしまいます。
